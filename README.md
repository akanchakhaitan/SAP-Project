About This Project
This project report provides a comprehensive, end-to-end documentation of the Record-to-Report (R2R) process with a focus on the Month-End and Year-End Financial Close cycle. R2R is a core Finance & Accounting process that covers financial data collection, processing, reconciliation, and reporting.
The report is submitted as part of the SAP Data Analytics course and demonstrates practical understanding of financial close processes as implemented in ERP environments such as SAP S/4HANA.
